Описание

bestcompile — это функциональный язык программирования, реализованный для учебных целей. Этот язык поддерживает основные возможности функционального программирования и предоставляет простой синтаксис для написания и выполнения программ.

Возможности

  • Именованные переменные (let)
  • Рекурсия
  • Лямбда-выражения (lambda)
  • Условные выражения (if)
  • Арифметические операции (+, -, *, /)
  • Функции печати (print)

Синтаксис

Числа

Числа представлены в виде целых или вещественных значений.

123
45.67

Строки

Строки заключаются в двойные кавычки.

"Hello, World!"

Логические значения

Поддерживаются логические значения true и false.

true
false

Идентификаторы

Идентификаторы представляют переменные и функции.

x
myFunction

Переменные

Для присваивания значений переменным используется ключевое слово let.

(let x = 10)

Условные выражения

Условные выражения используют синтаксис if.

(if (< x 10) "less" "greater or equal")

Лямбда-выражения

Лямбда-выражения объявляются с помощью ключевого слова lambda.

(lambda (x) (* x x))

Вызовы функций

Вызовы функций заключаются в круглые скобки.

(myFunction 5 10)

Пример программы: Факториал

(let factorial = (lambda (n) (if (<= n 1) 1 (* n (factorial (- n 1)))))) 
(factorial 5)

Пример программы: Объявление переменных

(let x = 10)
(let y = 20)
(print (+ x y))

Пример программы: Объявление функций

(let square = (lambda (x) (* x x)))
(print (square 5))

Пример программы: Счётчик с использованием замыкания

(let square = (lambda (x) (* x x)))
(print (square 5))

Установка и запуск

Требования

  • .NET SDK 6.0 или новее

Сборка проекта

  1. Клонируйте репозиторий: 
    git clone https://github.com/MrMazer/docks_best_compile
cd bestcompile/bestcompile
  2. Соберите проект: 
    dotnet build

Запуск программы

  1. Создайте файл с исходным кодом на bestcompile, например, example.bs. 
    (let factorial = (lambda (n) (if (<= n 1) 1 (* n (factorial (- n 1)))))) 
(print (factorial 5))
  2. Запустите программу: 
    dotnet run -- "example.bs"

Структура проекта

  • AST.fs: Модуль, содержащий определение абстрактного синтаксического дерева (AST).
  • Lexer.fs: Модуль, содержащий лексер для разбора исходного кода на токены.
  • Parser.fs: Модуль, содержащий парсер для построения AST из токенов.
  • Interpreter.fs: Модуль, содержащий интерпретатор для выполнения программ на bestcompile.
  • Program.fs: Главный файл программы, который связывает все модули и запускает интерпретатор.
